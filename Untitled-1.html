第二章 机器学习基础

2.1 机器学习概述

机器学习的本质：让机器模拟人类学习活动：学知识，用知识。具体而言，机器学习是让机器从已有数据中学习掌握知识，并能够识别现有知识的各种变体。

核心思想：见过的样本下次见到（不管怎么变化）还能认识，甚至举一反三。

机器学习基本概念

  • 数据集：多个对象或样本客观描述构成的集合，通常与所需学习任务直接相关
  • 样本：数据集中的单个实例
  • 特征向量：计算机中对某一对象属性的数值化表示（如：色泽=青绿 → 12）
  • 特征空间：将各特征属性视为坐标轴形成的坐标空间
  • 训练/学习：从数据集中学得模型的过程
  • 真值/标签 (Ground Truth/Label)：样本的真实类别或数值

常见学习任务

分类 (Classification)

给定某一数据，让机器学会对其某一属性正确分类。

输出：离散值（如：好瓜/坏瓜，0/1，A/B/C类别）

应用场景：

  • 机器人目标识别
  • 机器人文字阅读
  • 图像识别
  • 语言识别

回归 (Regression)

预测连续数值结果。

输出：某一区间的连续值（如：0-255的像素值，坐标位置）

应用场景：

  • 图像修复（输出像素值）
  • 目标定位（输出坐标）
  • 机器人视觉超分辨率

机器学习类型

学习类型 特点 常见任务 应用场景
有监督学习
(Supervised Learning)		 每个样本包含特征描述，且均对应有真值 分类、回归 图像识别、文字识别、目标定位
无监督学习
(Unsupervised Learning)		 每个样本仅有特征描述，没有真值 聚类、降维 数据标注、信息检索、可视化分析

无监督学习示例

聚类分析：当没有真值的时候，机器可以将相似的对象归到同一类。

特征降维：让机器学习剔除冗余的特征维度。

例如：3x3图像被转换为9维特征向量 (12,12,18,123,124,132,221,129,162)

数据集划分

训练集、验证集、测试集

  • 训练集：用于模型学习（构建模型）
  • 验证集：训练中途评估模型，调整超参数
  • 测试集：训练结束后评估模型泛化能力

数据集划分方法

方法 描述 特点
留出法 留出小部分数据（如20%）作为测试集 简单快速，但数据利用率低
K折交叉验证 将数据集划分为K个子集，每个子集轮流作为测试集 数据利用率高，结果更可靠
K折交叉验证过程：
数据集D = D₁ ∪ D₂ ∪ ... ∪ Dₖ
每次取Dᵢ作为测试集，其余作为训练集
最终结果取K次测试结果的平均值

模型评估

泛化能力 (Generalization)

机器学习模型对于同类型不同样本的通用能力。

过拟合 (Overfitting)

问题：学习了过多训练样本独有特点

表现：

  • 训练集误差小，测试集误差大
  • 死记硬背，只会做习题
  • 一到考试（新数据）就不行

欠拟合 (Underfitting)

问题：未能学习到训练集的一般规律

表现：

  • 训练集和测试集误差都大
  • 只学到皮毛、浅尝辄止
  • 模型过于简单

分类评估指标

混淆矩阵

真实情况 预测结果
正例 反例
正例 真正例 (TP) 假反例 (FN)
反例 假正例 (FP) 真反例 (TN)

关键指标

精度 (Accuracy) = (TP + TN) / (TP + TN + FP + FN)
查准率 (Precision) = TP / (TP + FP)
查全率 (Recall) = TP / (TP + FN)

查准率与查全率关系：

  • 追求查全率：宁可错杀（降低阈值，检出更多正例）
  • 追求查准率：不可滥杀无辜（提高阈值，只选确信的正例）
  • 平衡点：查全率 = 查准率

回归评估指标

均方误差 (MSE) = (1/m) * Σ(f(xᵢ) - yᵢ)²

其中：

  • m：样本总量
  • f(xᵢ)：模型预测值
  • yᵢ)：样本真实值

机器学习数学描述

y = f(x)

其中：

  • y：输出（标签）
  • f：学习函数（模型）
  • x：特征描述（输入）

训练过程

给定训练集 {(x₁, y₁), ⋯, (x_N, y_N)}，从中学习特征x至标签y的映射函数f，使得函数输出匹配训练样本。

测试过程

将学习函数f应用到训练集中未曾见过的样本x，得到预测输出y = f(x)。
